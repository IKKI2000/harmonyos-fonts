## 🌟 特点

* 基于官方 HarmonyOS Sans
* 支持中文、拉丁文、西里尔文、希腊文、阿拉伯文语系下的105种语言
* 语言、文字的字体自适应
* 字体的字重、字形自适应
* 多种使用方式
